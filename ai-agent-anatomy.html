0 · More Than the Model

An AI agent is not just the AI model. The model is one component inside a classic software system that stores context, decides what data the model can see, validates proposed actions, calls real APIs, handles failures, and records what happened.

AI model helps decide

  • Interprets the user goal and the current context.
  • Reasons, plans, drafts, and chooses a likely next step.
  • Proposes structured tool calls such as draft_refund(...).

Classic software controls

  • Builds context from databases, files, search, and prior state.
  • Checks schemas, permissions, approval rules, and policy boundaries.
  • Executes APIs, manages retries and timeouts, logs the audit trail.
